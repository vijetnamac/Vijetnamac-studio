## 🚀 Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `vijetnamac-studio`)
2. Push this folder:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/vijetnamac-studio.git
git push -u origin main
```
3. Go to **Settings → Pages → Source: main / root** → Save
4. Your site will be live at `https://YOUR_USERNAME.github.io/vijetnamac-studio/`
